MDBs_HW3_Code 
This is a homework assignment to demonstrate how to connect to postgresql relational database to an local application. 
The program is written in java. 

To execute the program the user will have to use the IntelliJ IDEA. 
I tried to use javac and java in command prompt but ran into some issue with linking the the database driver. 

To run the code, the user will open the folder as a IntelliJ java project. 
Add the newest JDK for compiling the code. 
In library add JDBC driver.
